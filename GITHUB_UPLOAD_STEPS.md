# How to Upload to GitHub

1. Create a new empty GitHub repository.
2. Open Command Prompt inside this folder.
3. Run:

```cmd
git init
git add .
git commit -m "Complete PopEyez Milestone 2 full-stack project"
git branch -M main
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```

Each team member should make at least one meaningful commit, for example editing README.md, adding tests, improving CSS, or updating seed data.
