PopEyez Milestone 2 - GitHub Ready Full-Stack Project

1) Extract the ZIP file.
2) Open the extracted folder: PopEyez_Milestone2_GitHubReady
3) Double-click START_BACKEND.bat and wait until it says backend running on http://localhost:5000
4) Double-click START_FRONTEND.bat and open the shown Vite URL, usually http://localhost:5173

If double-click does not work, use Command Prompt:
cd /d "C:\path\to\PopEyez_Milestone2_GitHubReady\backend"
npm install
npm run seed
npm start

Then in another Command Prompt:
cd /d "C:\path\to\PopEyez_Milestone2_GitHubReady\frontend"
npm install
npm run dev
