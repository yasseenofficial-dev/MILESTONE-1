# PopEyez User Journey Coverage Matrix

This project implements all 26 journeys from the Milestone 2 User Journey Documentation.

| Journey | User | Implemented screens / APIs |
|---|---|---|
| 1 Account Creation and Customization | Organizer | Accounts page, `/api/users` create/update/deactivate, login |
| 2 Venue Search and Booking | Organizer | Venues page, filters, booking application API |
| 3 Event Planning & Setup | Organizer | Events/tasks, budget, layout designer, staff assignment |
| 4 Vendor Coordination | Organizer | Vendor list, sourcing requests, delivery status, invoice review |
| 5 Guest Management | Organizer | Guest list, invitation sending, RSVP and dietary tracking |
| 6 Day-Of Operations | Organizer | Operations dashboard, live messages, seen/unseen follow-up |
| 7 Evaluation and Reporting | Organizer | Reports page, feedback analytics, cost/attendance CSV export |
| 8 Profile Access | Staff | Staff login through seeded credentials |
| 9 Task Management & Execution | Staff | My events, my tasks, status/progress update |
| 10 Layout Utilization | Staff | Shared floor plan viewer |
| 11 Day-Of Logistics | Staff | Guest check-in and vendor arrival coordination |
| 12 Events Dashboarding | Staff | Staff operations dashboard with guest and task context |
| 13 Account Access | Vendor | Vendor login/register profile style update |
| 14 Sourcing Request Management | Vendor | Incoming requests, details, accept/decline |
| 15 Delivery Management | Vendor | Accepted orders, delivery status updates, arrival confirmation |
| 16 Invoicing | Vendor | Invoice submission and invoice status tracking |
| 17 Invitation Receipt | Guest | Invitation page with event details |
| 18 RSVP | Guest | RSVP status, dietary preferences, special requirements |
| 19 Day-Of Communications | Guest | Guest messages and seen status |
| 20 Event Check-In | Guest | QR/name confirmation page and check-in status |
| 21 Post-Event Feedback | Guest | Feedback form and thank-you acknowledgment |
| 22 Account Setup | Venue Owner | Venue owner login and profile update |
| 23 Venue Listing Management | Venue Owner | Create/deactivate listings and view listing details |
| 24 Booking Request Management | Venue Owner | Approve/decline/counter booking requests |
| 25 Booking Overview | Venue Owner | Confirmed booking list |
| 26 Performance & Reporting | Venue Owner | Venue dashboard and CSV export |
