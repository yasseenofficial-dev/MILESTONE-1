# Testing Checklist

## Startup
- [ ] Run `npm install` in backend.
- [ ] Run `npm run seed` in backend.
- [ ] Run `npm start` in backend.
- [ ] Run `npm install` in frontend.
- [ ] Run `npm run dev` in frontend.
- [ ] Open `http://localhost:5173`.

## Organizer Tests
- [ ] Log in as `organizer@popeyez.com`.
- [ ] Open Day-Of Dashboard.
- [ ] Change selected event.
- [ ] Confirm total guest and arrived guest numbers appear.
- [ ] Confirm vendor and task progress appears.
- [ ] Open Communications.
- [ ] Send a new day-of message.
- [ ] Confirm it appears in communication history.
- [ ] Trigger a follow-up for unseen guests.
- [ ] Open Reports.
- [ ] View attendance, cost, task, vendor, and feedback results.
- [ ] Export CSV.

## Staff Tests
- [ ] Log in as `staff@popeyez.com`.
- [ ] Open My Events.
- [ ] Filter by date.
- [ ] Open My Tasks.
- [ ] Filter by status.
- [ ] Update a task status/progress.
- [ ] Open Floor Plan and view layout.
- [ ] Open Guest Check-In.
- [ ] Filter by check-in status.
- [ ] Mark a guest as checked in.
- [ ] Open Vendor Arrivals.
- [ ] Mark a vendor as arrived.
